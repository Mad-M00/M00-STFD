# Community page assets — 7daystodiemods.com / Nexus

Files for publishing STFD on mod sites:

| File | Use |
|---|---|
| `title.txt` | Page title |
| `homepage.bbcode.txt` | Main mod page body (BBCode) |
| `install.bbcode.txt` | Installation section (BBCode) |
| `changelog.bbcode.txt` | Version history (BBCode) |
| `promo-art.png` | Full-size key art for the mod page |
| `thumbnail-400x225.png` | Listing thumbnail |
| `banner-prompt.txt` | The AI prompt that generated the art (for reruns) |

The repo banner (`docs/images/banner.png`) is the same art; the homepage
BBCode hotlinks it from raw.githubusercontent.com.

Keep the two "Straight answers" entries on the page — they pre-empt the
most common comment-section questions (sleeper-room wakes, quest-door
picking). Update `changelog.bbcode.txt` with every release.
